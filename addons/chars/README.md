# Chars #
This addon lets you input special chars using simple tags (ex.: ```<note>``` for â™ª). Using the pattern ```<j:text>``` any alphanumeric character will be replaced with its japanese version. The available characters depend on the data gathered by the windower team (https:_github.com/Windower/Lua/blob/master/addons/libs/ffxidata.json)

![screenshot](http://i39.tinypic.com/spdwz6.png) 

## Commands ##
### chars ###
Shows the available characters

```
chars 
```
