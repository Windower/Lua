Author: Ricky Gall

Version: 1.07

Addon to enable 1 macro status removal with the help of send.

Abbreviation: //stna

Using //stna will tell your mule to remove status effects in the following Default order:
1. Doom
2. Curse
3. Petrification
4. Paralysis
5. Plague
6. Silence
7. Blindness
8. Poison
9. Diseased
